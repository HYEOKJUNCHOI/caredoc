# PROJECT.md — CareDoc (케어독)

## ★ 1. 프로젝트 개요
- **프로젝트명:** CareDoc (케어독)
- **한줄 요약:** 일본 장애인 그룹홈 시설 담당자가 반복 작성하는 복지 서류(개별지원계획서·모니터링기록표·작성회의록)를 버튼 클릭만으로 빠르게 완성하고 Word/PDF로 출력할 수 있는 웹앱.
- **버전:** v1.0 MVP (Minimum Viable Product)

---

## ★ 2. 설계 철학
- **Click, not type:** 담당자가 타이핑하는 시간을 없앤다. 자주 쓰는 문구는 버튼 하나로 입력된다.
- **의사소통이 어려운 이용자도 빠짐없이:** 본인 감상·의견란처럼 직접 듣기 어려운 항목도 상황별 선택지를 제공해 공백 없이 작성할 수 있게 한다.
- **파일 하나, 링크 하나:** 설치 없이 URL 접속만으로 사용. 비IT 사용자도 부담 없이 쓸 수 있어야 한다.

---

## ★ 3. 타겟 유저
- **주요 사용자:** 일본 장애인 그룹홈 시설의 생활지원원 담당자 (현재는 1인 사용 기준)
- **IT 숙련도:** 스마트폰·PC 기본 사용 가능, 전문 소프트웨어 경험 없음
- **사용 환경:** 사무실 PC 또는 태블릿, 브라우저 접속
- **핵심 불편:**
  - 3종 서류에 동일한 이용자 정보·지원목표를 매번 따로 입력
  - 의사소통이 어려운 이용자의 감상·의견란을 매번 머릿속에서 짜내야 함
  - Excel/Word 파일을 열고 수동으로 채우는 반복 작업


## ★ 11. 가족관계도 기능 상세

### 개요
이용자 등록 폼에서 가족 구성원을 체크하면 SVG로 가족관계도가 실시간으로 그려지고,
서류 출력(Word/PDF/프린트) 시 개별지원계획서 하단에 자동 포함된다.

### 가족 구성원 목록 (체크박스)

| 관계 | 한국어 | 일본어 |
|------|--------|--------|
| 부 | 아버지 | 父 |
| 모 | 어머니 | 母 |
| 형 | 형 | 兄 |
| 제 | 남동생 | 弟 |
| 자 | 누나/언니 | 姉 |
| 매 | 여동생 | 妹 |
| 조부 | 할아버지 | 祖父 |
| 조모 | 할머니 | 祖母 |
| 배우자 | 배우자 | 配偶者 |
| 자녀 | 자녀 | 子 |
| 기타 | 기타 (직접입력) | その他 |

### SVG 관계도 레이아웃 규칙

```
레이아웃: 이용자(본인)를 중앙에 고정, 관계별 위치 규칙:

          [조부] [조모]
               |
          [부] [모]
          /         \
    [형/제]  ★본인★  [자/매]
                |
           [배우자]
                |
            [자녀]
```

- 이용자(본인) 노드: 원형, `--color-primary` 배경, 굵은 테두리
- 가족 노드: 둥근 사각형, `--color-secondary` 배경
- 연결선: `--color-border` 색상의 직선
- 체크된 가족만 노드 표시, 미체크 시 해당 노드·선 숨김
- SVG 크기: 너비 100%, 높이 자동 조정

### 출력 포함 방식
- **미리보기(Preview):** 개별지원계획서 미리보기 하단에 FamilyDiagram 컴포넌트 삽입
- **PDF:** html2canvas가 SVG 포함 DOM을 그대로 캡처 → 별도 처리 불필요
- **Word:** docx 라이브러리로 SVG → PNG 변환 후 이미지로 삽입 (추후 구현)
- **프린트:** CSS `@media print`에서 SVG 영역 표시
